## **Introduccion**
Este proyecto es un Frontend moderno desarrollado con React + Vite que se integra con un Backend en Flask. La aplicación está diseñada para ofrecer una interfaz de usuario interactiva y receptiva, utilizando tecnologías web modernas.

Características principales:
Arquitectura: Aplicación de una sola página (SPA) con React
Estilización: Utiliza módulos CSS para un diseño modular y mantenible
Componentes principales:
- Páginas de autenticación (Login)
- Sección de Blog con detalle de publicaciones
- Página "Acerca de" con información del proyecto
- Componente de pie de página (Footer)
Tecnologías utilizadas:
- Frontend: React 18, Vite, React Router
- Backend: Python con Flask
- Base de datos: MySQL
- Herramientas de desarrollo: ESLint, Git

## Guia de instalacion

1. clonar el repositorio <br/>
**git clone git@github.com:Emancc/EFI-Front-Api.git**

## Comandos de inicialización del proyecto

1. npm install
2. npm run dev

## Conección con Backend
1. clonar el repositorio <br/>
**git clone git@github.com:Emancc/EFI-Backend-Api.git**

## Comandos de inicializacion del Backend
1. crear entorno virtual 
- en linux <br/>**python -m venv venv**
- en windows <br/>**py -m venv venv**
- en mac <br/>**python3 -m venv venv**

2. activar entorno virtual
- en linux <br/>**source venv/bin/activate**
- en windows <br/>**venv\Scripts\activate**
- en mac <br/>**source venv/bin/activate**

3. instalar dependencias
- en linux <br/>**pip install -r requirements.txt**
- en windows <br/>**pip install -r requirements.txt**
- en mac <br/>**pip install -r requirements.txt**

4. iniciar servidor
- en linux <br/>**flask run --reload**
- en windows <br/>**flask run --reload**
- en mac <br/>**flask run --reload**
